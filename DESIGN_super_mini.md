# Design System
**Theme:** light
**Colors:**
- Primary Text / Midnight Ink: `#141414`
- Page Background / Pure Canvas: `#ffffff`
- Body Copy / Graphite: `#707070`
- Border / Fog: `#ededed`
- Inner UI / Mist: `#f2f2f2`

**Typography:**
- Font Family: `saans`, Inter Variable, sans-serif
- Weights: 400, 440, 456, 600, 652

**Shapes:**
- Pill Elements (buttons, inputs, tags): `9999px`
- Containers (cards, thumbnails): `16px` to `24px`

**Shadows:**
- Flat: 1px `#ededed` border. Dropdowns use `rgba(0,0,0,0.04) 0px 8px 40px`.
